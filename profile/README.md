## 👋 Tentang Kami

Selamat datang di _organization profile_ resmi GitHub milik **Instalasi Teknologi Informasi RSUD Blambangan (IT RSBL)**, Banyuwangi. Kami adalah tim pengembang perangkat lunak, _system administrator_, dan teknisi jaringan yang berfokus pada digitalisasi layanan kesehatan di Rumah Sakit Umum Daerah Blambangan.

Kami berkomitmen untuk menghadirkan solusi teknologi informasi yang andal dan aman demi mendukung tenaga medis, manajemen rumah sakit, serta memberikan pengalaman pelayanan terbaik dan tercepat bagi pasien.

## 🚀 Misi Kami

- 💻 **Digitalisasi Layanan:** Membangun dan mengembangkan sistem pendaftaran, pelayanan, dan _back-office_ secara digital yang terintegrasi.
- 💡 **Inovasi Berkelanjutan:** Konsisten mengadopsi teknologi terbaru untuk optimalisasi _flow_ bisnis RSUD Blambangan.
- 🔗 **Mendukung Terbentuknya Ekosistem Kesehatan Nasional:** Implementasi _bridging_ (integrasi API) ke Kemenkes (SatuSehat) dan BPJS Kesehatan (VClaim, Antrean Online, Mobile JKN, dll).
- 🔒 **Keamanan & Transparansi Data:** Menjamin berjalannya _paperless hospital_ melalui Rekam Medis Elektronik (RME) yang aman, cepat, dan _reliable_.
- ⚙️ **Ketersediaan 24/7:** Memastikan seluruh server, infrastruktur jaringan ruang rawat, IGD, dan ekosistem IT rumah sakit dapat beroperasi terus menerus.

## 🛠️ _Tech Stack_ Internal

Dalam proses _development_ dan pemeliharaan _server_, IT RSBL memanfaatkan berbagai teknologi mulai dari bahasa pemrograman modern hingga infrastruktur terpusat:

- **Backend:** PHP (Laravel), Node.js
- **Frontend:** HTML/CSS/JS, Vue.js, React, Bootstrap, Tailwind CSS, Alpine.js
- **Database:** MySQL
- **Infrastruktur & Jaringan:** Mikrotik, Cisco, Linux Server (Ubuntu), Proxmox VE, Docker, Nginx/Apache
